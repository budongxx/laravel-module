## 介绍

houdunwang/laravel-module 是一种laravel包，它通过使用模块来管理大型laravel应用程序。模块就像一个laravel包，它有一些视图、控制器或模型。这个包在Laravel 5中得到支持和测试。
这个包在 HDCMS 中使用。
